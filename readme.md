Créer un dossier "web4heroes" dans devilbox > data > www > web4heroes
Mettre un dossier htdocs dans le dossier web4heroes
Ouvrir un terminal sur ce fichier htdocs
entrez :          git clone https://github.com/Velskins/web4heroes.git
Allez sur votre phpMyAdmin de devilbox ( Tools -> PhpMyAdmin )
Créer une nouvelle base de donnée et allez dans SQL dans le menu de nav en haut
Ouvrez sur votre IDE le dossier htdocs et copiez le fichier web4heroes.sql qui se trouve dans Config > web4heroes.sql
Exécutez la création.

Maintenant nous avons tous la même base
